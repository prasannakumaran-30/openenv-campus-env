# Campus Operations OpenEnv

A real-world simulation environment where AI agents manage college scheduling under dynamic disruptions.

## Features
- Multi-step decision making
- Dynamic disruptions (faculty unavailable, room conflicts, new classes)
- Priority-based reward system
- Easy, Medium, Hard task levels

## Actions
- assign_room
- reschedule
- cancel

## Observations
- class schedules
- priorities
- disruptions
- step count

## Run Locally
